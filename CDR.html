SafeX

Helping Riders Stay Safe

Problem Statemnt

Majority of products for motorcyclists are tailored for convenience rather than safety. Typically riders must manually check their blindspots via side-mirrors and shoulder checks. This becomes a hazard for riders as they become distracted when doing so. After vehicle accidents, there is no efficient way contacting emergency services in products on the market. Our solution will address these issues by notifying riders of necessary information while on the road and providing post-crash safety measures to better their safety.

Intro to SafeX

Project Goal

There are alot of safety features available for cars that are not accessible to motorcycles. Additionally, there are also missing features that should be available to motorcyclists. We hope to bring some of those safety features to motorcycles in order to ensure the safety of riders.

To create a two part device that is attachable to a motorcycle and helmet. The system should notify riders of oncoming vehicles within their blindspots to increase rider safety, and alert emergency services if the rider gets into a crash.

Blindspot Detection:

tl;dr: This demo shows the Blindspot Detection system which for demo purpose is set to detect both vehicles and people. When an object is detected in the blindspot of the motorcycle the object is classified accordingly and a bounding box is placed around them. The system then tracks the object's movement inorder to identify what side it is relative to the rider. This information is then sent over to the helmet which uses the LEDs to notify where the object is located.

tl;dr: This demo shows the Fall Detection system which monitors the rider's acceleration in the Y direction. When the acceleration exceeds a threshold (For demo purposes the threshold is set to a low value) the system will trigger the fall protocol where the LEDs on the helmet light up red to indicate to the rider that the protocol has been triggered. The helmet then communicates with the mobile app which then triggers a popup that allows the rider to select between two options. The rider can indicate to the system that they are safe to cancel the protocol. If they require medical assistance they can indicate that to the system using the "Call Now" button. If the countdown expires then it automatically contacts emergency services.
